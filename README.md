# Сердце в переплёте — сайт любовных романов

## Структура проекта

```
siteboock/
├── server.js           — Node.js сервер (Express + SQLite)
├── package.json        — зависимости
├── data.db             — база данных SQLite (создаётся автоматически)
├── index.html          — главная страница
├── style.css           — стили сайта
├── script.js           — логика фронтенда (загрузка из API)
├── admin.html          — админ-панель
├── admin.css           — стили админки
├── admin.js            — логика админки (CRUD через API)
└── assets/
    ├── favicon.svg
    ├── book1.jpg.svg   — анимированная обложка
    ├── book2.jpg.svg
    ├── book3.jpg.svg
    └── book4.jpg.svg
```

## Авторизация

Чтобы изменить, задайте переменные окружения перед запуском:
```bash
set ADMIN_LOGIN=мой_логин
set ADMIN_PASSWORD=мой_пароль
npm start
```

Или отредактируйте значения прямо в `server.js` (строки `ADMIN_LOGIN` и `ADMIN_PASSWORD`).

Без авторизации:
- Главная страница и чтение книг/блога доступны всем
- Админ-панель показывает страницу входа
- API на запись (POST/PUT/DELETE) возвращает 401

## База данных

Используется SQLite (через sql.js — без нативных зависимостей).
Файл `data.db` создаётся автоматически при первом запуске и заполняется дефолтными данными.

Все изменения через админку сохраняются в БД и видны всем посетителям сайта.
