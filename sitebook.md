### 3. Деплой на Render

1. Зайдите на [render.com](https://render.com) и зарегистрируйтесь (можно через GitHub)
2. Нажмите **New → Web Service**
3. Подключите ваш GitHub-репозиторий `siteboock`
4. Render автоматически определит настройки из `render.yaml`, но проверьте:
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
   - **Plan:** Free
5. В разделе **Environment Variables** добавьте:
   - `ADMIN_LOGIN` = ваш логин
   - `ADMIN_PASSWORD` = ваш пароль
6. Нажмите **Create Web Service**

Через 2-3 минуты сайт будет доступен по адресу вида:
```
https://heart-in-book.onrender.com
```

### Важные моменты

- **Засыпание:** На бесплатном плане Render сервис засыпает через 15 минут без запросов. Первый визит после сна занимает ~30-60 секунд. Для личного сайта это нормально.
- **БД:** При каждом новом деплое `data.db` пересоздаётся. Если хотите сохранять данные между деплоями — подключите [Render Disk](https://render.com/docs/disks) (бесплатно 1 ГБ) и измените `DB_PATH` на путь к диску.
- **Обновление сайта:** Просто пушьте изменения в GitHub — Render автоматически пересоберёт и задеплоит.
